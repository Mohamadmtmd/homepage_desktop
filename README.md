# Shamsipour Security - Desktop-First Landing Page

## Overview
This is a **desktop-first responsive landing page** for a fictional IT security company called Shamsipour Security. The design focuses on **desktop usage**, providing a professional layout with clearly organized sections, while still remaining responsive for smaller screens. The page is built using only **HTML, CSS, and JavaScript**, without any external frameworks.

## Features & Design Decisions

- **Desktop-First Design:** The layout is optimized for desktop viewports, with horizontal card arrangements and spacious sections.
- **Responsive Layout:** The page adapts to smaller screens with media queries.
- **Header & Navigation:** Includes brand logo/name and a navigation menu linking to each section. Active link highlighting on scroll.
- **Hero Section:** Large title, short description, and a call-to-action button.
- **Services Section:** Cards with **images, titles, and short descriptions** to showcase key services.
- **Projects Section:** Cards displaying project titles and descriptions, using fictional companies (Joya, Sorena, Arman) with relevant data center images.
- **Pricing Plans:** Three pricing tiers with feature highlights.
- **Testimonials / Clients:** Section reserved for quotes or logos (can be expanded in future).
- **Footer:** Contact info, social media links, and copyright information.
- **Interactive Elements:** 
  - Scroll-based active navigation link highlighting
  - Back-to-top button for easy navigation

## Technologies Used

- **HTML5**: Semantic structure of the page
- **CSS3**: Styling, Flexbox for layout, responsive media queries
- **JavaScript**: Hamburger menu toggle, scroll detection for active links, back-to-top button

## Color Palette & Typography

- **Background:** Dark (#0c0c0c)
- **Primary Accent:** Blue (#00d1ff) for buttons, card highlights, and active links
- **Text:** Light (#eaeaea) for readability
- **Font:** IRANSans (or system sans-serif fallback)
